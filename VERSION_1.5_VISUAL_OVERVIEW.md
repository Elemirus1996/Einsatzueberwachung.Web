## ?? Verwendungs-Beispiele

### Beispiel 1: Vermisstensuche im Wald
```
ELW (Einsatzleiter):
  • Laptop mit Desktop-Version
  • Erstellt 3 Hunde-Teams
  • Startet Timer
  • Vergibt Suchgebiete

Gruppenführer (im Wald):
  • Tablet mit Mobile Dashboard
  • Überwacht alle Teams
  • Sieht Timer-Warnungen
  • Liest Funksprüche

Hundeführer (im Einsatz):
  • Smartphone mit Mobile Dashboard
  • Sieht eigenes Team
  • Sieht andere Teams
  • Kann Funksprüche lesen
```

### Beispiel 2: Großübung
```
Übungsleiter:
  • Desktop-Version
  • 8 Teams gleichzeitig
  • Live-Monitor auf Beamer

Beobachter:
  • 5 Tablets mit Mobile Dashboard
  • Verteilte Positionen
  • Alle sehen alles live
  • Keine Funkgeräte nötig
```

## ? Besonderheiten

### Was macht v1.5.0 besonders?

1. **Null-Konfiguration für Mobile**
   - QR-Code scannen, fertig!
   - Kein App-Download nötig
   - Kein Account erforderlich

2. **Echtzeit ohne Polling**
   - SignalR = echtes Push
   - Keine Verzögerung
   - Minimaler Akku-Verbrauch

3. **Funktioniert offline (Desktop)**
   - Desktop-Version läuft ohne Internet
   - Daten lokal gespeichert
   - Nur Mobile benötigt Netzwerk

4. **Open Source**
   - Vollständiger Quellcode
   - MIT-Lizenz (zu klären)
   - Community-Contributions willkommen
