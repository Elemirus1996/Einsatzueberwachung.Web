## ?? Quick Start Guide

### 1. Installation
```bash
git clone https://github.com/Elemirus1996/Einsatzueberwachung.Web.git
cd Einsatzueberwachung.Web
git checkout v2.5.0
dotnet restore
```

### 2. Starten
```bash
# Einfach
cd Einsatzueberwachung.Web
dotnet run

# Mit Netzwerk
START_NETWORK_SERVER.bat
```

### 3. Dark Mode testen
```
1. Browser öffnet automatisch
2. In Navigation auf ?? Icon klicken
3. Theme wechselt zu Dark Mode
4. Neuen Tab öffnen
5. Theme ist synchronisiert! ?
```
