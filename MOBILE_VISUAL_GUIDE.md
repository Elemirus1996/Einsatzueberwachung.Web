## ?? Farb-Schema

### Status-Indikatoren

```
Team-Status:
?? ?? GRÜN   = Im Einsatz, keine Warnung
?? ?? GELB   = Erste Warnung (45+ Min)
?? ?? ROT    = Zweite Warnung (60+ Min, blinkt)
?? ? GRAU   = Bereit (nicht gestartet)

Timer-Farben:
?? Grün   = Aktiv, normal
?? Orange = Erste Warnung
?? Rot    = Zweite Warnung (blinkt)

Team-Karten Hintergrund:
?? Grau-Gradient    = Bereit
?? Grün-Gradient    = Aktiv
?? Orange-Gradient  = Warnung
?? Rot-Gradient     = Kritisch (pulsiert)
```
