## ?? Tipps & Tricks

### ?? Tipp 1: Icon ändern
```
1. Rechtsklick auf Desktop-Shortcut
2. Eigenschaften
3. "Symbol ändern..."
4. Eigenes .ico auswählen
5. OK
```

### ?? Tipp 2: Konsolenfenster verbergen (Fortgeschritten)
```
1. Rechtsklick auf Shortcut ? Eigenschaften
2. "Ausführen:" ? "Minimiert"
3. OK
```

### ?? Tipp 3: Mehrere Versionen parallel
```
- Desktop-Shortcut 1: HTTPS (Standard)
- Desktop-Shortcut 2: HTTP (Backup)
- Desktop-Shortcut 3: Menu (Admin)
```
