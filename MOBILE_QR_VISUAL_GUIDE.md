# ?? QR-Code Mobile Zugriff - Visuelle Anleitung

## ?? Schnell-Übersicht

```
Server starten ? QR-Code anzeigen ? Scannen ? Fertig!
   (1 Min)          (am Desktop)       (Mobile)  (Auto)
```

## ?? 3-Schritte-Anleitung

### Schritt 1: Server starten
```cmd
START_NETWORK_SERVER.bat
```

### Schritt 2: QR-Code-Seite öffnen
```
http://localhost:5000/mobile/connect
```

### Schritt 3: Mit Smartphone scannen
- Kamera-App öffnen
- QR-Code scannen
- Fertig!
